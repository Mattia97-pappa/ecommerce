 Online Clothing Store 👕🛍️

Full-stack web application for an online clothing store, developed in Java.

Features

- User authentication
- Product browsing
- Order placement
- Data persistence

 Technologies

- Java
- Spring Framework
- Spring Security
- JPA (Hibernate)
- Bootstrap
- JavaScript
